EE244: Mid-term Project

Adrian Wrixon and Eugene Teh
 
November 1997

 

The description of this project was to implement two routers, a channel router and a switchbox router, and to demonstrate the operation of each on two given test examples. The algorithms we have written are designed to be general and should perform correctly on routing problems of greater complexity than those provided.
